# Machine Learning Roadmap

------------------------------------------------------------------------

## Weekly Study Structure (7-Day Cycle)

-   **Day 1--2 (Learn):**
    -   Study concepts (videos, docs, notes)
-   **Day 3--4 (Practice):**
    -   Solve problems, implement small code snippets
-   **Day 5--6 (Build):**
    -   Apply learning in a mini-project
-   **Day 7 (Review):**
    -   Revise notes, fix mistakes, document learning

------------------------------------------------------------------------

## Rules to Follow

-   Avoid tutorial-only learning
-   Always build something after learning
-   Stay consistent \> being perfect
-   Focus on understanding, not memorizing
-   Document everything on GitHub

------------------------------------------------------------------------

# Stage 1: Foundations (Weeks 1--4)

## Goal

Build strong basics in Python, math, and data handling.

## Topics

-   Python (NumPy, Pandas)
-   Basic Statistics (mean, variance, distributions)
-   Linear Algebra (vectors, matrices)
-   Data visualization (Matplotlib, Seaborn)

## Practical Tasks

-   Load and analyze datasets
-   Perform basic data cleaning
-   Plot graphs

## Mini Projects

-   Exploratory Data Analysis (EDA) on any dataset
-   Data visualization dashboard

## Checklist

-   [ ] Understand NumPy arrays
-   [ ] Use Pandas DataFrames
-   [ ] Plot graphs
-   [ ] Basic statistics understanding

------------------------------------------------------------------------
